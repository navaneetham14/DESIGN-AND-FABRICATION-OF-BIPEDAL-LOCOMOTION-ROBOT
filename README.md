# 🤖 Design and Fabrication of Bipedal Locomotion Robot

## 📖 Overview

The **Design and Fabrication of Bipedal Locomotion Robot** is a robotics project developed to imitate human walking using a two-legged robotic mechanism. The robot is designed with a lightweight mechanical structure and controlled using an **ESP32 microcontroller** and **PCA9685 PWM servo driver**. It performs predefined walking motions such as forward, backward, and turning while maintaining balance.

This project was developed as a **Mini Project** for the Bachelor of Engineering in Robotics and Automation. :contentReference[oaicite:0]{index=0}

## ⚙️ Working Principle

The robot is powered by a **7.4V Lithium-Ion battery**, whose output is regulated using an **LM2596 Buck Converter**. The **ESP32** acts as the main controller and communicates with the **PCA9685 PWM servo driver** via the **I²C protocol**. The servo driver controls four **MG996R servo motors**, enabling coordinated hip and knee movements for stable walking, balancing, and turning operations. :contentReference[oaicite:4]{index=4}
