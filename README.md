# DESIGN-AND-FABRICATION-OF-BIPEDAL-LOCOMOTION-ROBOT
**Bipedal Locomotion Robot** is a two-legged robot that mimics human walking using servo motors, an ESP32 microcontroller, and a lightweight frame. It performs programmed movements like walking and turning while demonstrating robot mechanics, motion control, and balance. It serves as a foundation for humanoid robotics research and education.
